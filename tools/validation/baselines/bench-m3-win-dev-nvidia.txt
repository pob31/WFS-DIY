#

#   for Ir; WFS_GPU_HOST_WORKERS overrides (0 = sequential kill switch). On THIS

#   numbers were captured with WFS_GPU_HOST_WORKERS=3 (3 workers + the pump =

#   ONE backend synchronously, so it never shows the target rig's win from

#   run-to-run noise. The 64x128 rig shape (run 2) has ~8x the per-input prep
